# `movement` presentation at CBIAS 2025

This repository contains the slides for the **"Movement: a Python toolbox for analysing motion tracking data"** talk, presented by [Niko Sirmpilatze](https://github.com/niksirbi) at the [Crick Bioimage Analysis Symposium (CBIAS) 2025](https://www.crick.ac.uk/whats-on/crick-bioimage-analysis-symposium-2025).

You can view the slides online at [neuroinformatics.dev/slides-movement-cbias2025](https://neuroinformatics/slides-movement-cbias2025/#/title-slide).
